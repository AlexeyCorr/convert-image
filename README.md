# Image converter

 _Image compressor and converter to webp and avif format_


## Quick Start

1. `npm ci`
2. Place the images (jpg/png) in a folder named image
3. Place the images (svg) in a folder named svg
4. `npm start`


### Features

- [x] imagemin
- [x] svgo
- [x] toWebp
- [x] toAvif
- [ ] resize
- [ ] toWebm
